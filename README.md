## 📋 기능 구현

### Authentication

#### ① 회원가입
  - 입력값 중복 및 유효성 검사
  - 닉네임 랜덤 생성 버튼
#### ② 로그인
   - BCrypt 암호화
   - 쿠키 기반 JWT 인증

### MyPage
   - 로그인 여부에 따른 UI 노출
   - 회원 정보 확인
   - 회원 탈퇴

### Admin
   - 권한에 따른 UI 노출
   - 회원 목록 확인

### Board
#### ① 자유 게시판
   - 권한에 따른 글 생성/읽기/수정/삭제
   - 권한에 따른 댓글 생성/읽기/수정/삭제
   - 글 작성자가 댓글 작성시 "글쓴이" 표시
   - 조회수
   - 검색/페이징

#### ② 공지사항 게시판
   - 권한에 따른 글 생성/읽기/수정/삭제
   - 조회수
   - 검색/페이징
